Informe de Ejecución de Prueba

Caso de Prueba: Iniciar Sesión Exitosamente

Resultado: Exitoso

Descripción: El usuario pudo iniciar sesión satisfactoriamente, ingresando credenciales válidas.

Criterios de Aceptación:

  • El usuario puede ingresar sus credenciales y hacer clic en el botón de inicio de sesión.
  • Después de iniciar sesión, el usuario debe ser redirigido a la página principal.

Caso de Prueba: Validación de Campos Obligatorios en el Formulario de Inicio de Sesión

Resultado: Exitoso

Descripción: El sistema mostró correctamente el mensaje de error al intentar enviar el formulario sin completar los campos obligatorios.

Criterios de Aceptación:

  • Los campos de usuario y contraseña son obligatorios.
  • Se muestra un mensaje de error si se intenta enviar el formulario sin completar los campos requeridos.

Caso de Prueba: Recuperar Contraseña

Resultado: Exitoso

Descripción: El usuario pudo recuperar su contraseña correctamente siguiendo el enlace de recuperación.

Criterios de Aceptación:

  • Existe un enlace para recuperar la contraseña en la página de inicio de sesión.
  • Al hacer clic en el enlace, el usuario es redirigido a una página donde puede ingresar su dirección de correo electrónico para restablecer la contraseña.

Caso de Prueba: Cerrar Sesión

Resultado: Exitoso

Descripción: El usuario pudo cerrar sesión satisfactoriamente después de haber iniciado sesión previamente.

Criterios de Aceptación:

  • Después de iniciar sesión, el usuario puede cerrar sesión haciendo clic en un enlace o botón correspondiente.
  • Después de cerrar sesión, el usuario debe ser redirigido a la página de inicio de sesión.

Caso de Prueba: Cambiar Contraseña

Resultado: Fallido

Descripción: La prueba para cambiar la contraseña falló al intentar cambiar la contraseña sin proporcionar la contraseña actual.

Criterios de Aceptación:

  • Después de iniciar sesión, el usuario puede acceder a una página para cambiar su contraseña.
  • El usuario debe proporcionar la contraseña actual y la nueva contraseña, y al enviar el formulario, la contraseña debe actualizarse correctamente.
